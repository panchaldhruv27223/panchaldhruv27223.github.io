# Dhruv Panchal - Portfolio Website

A modern, interactive portfolio website showcasing my work as a Machine Learning researcher and engineer. Built with HTML, CSS, and JavaScript, featuring smooth animations, 3D backgrounds, and a responsive design.

### Sections

- **Home**: Hero section with introduction and call-to-action buttons
- **About**: Personal background and achievements
- **Education**: Academic qualifications and achievements
- **Experience**: Professional work experience
- **Skills**: Technical skills organized by category
- **Projects**: Interactive project cards with detailed descriptions
- **Certificates**: Professional certifications with verification links
- **Blogs**: Links to technical blog posts
- **Contact**: Social media links and contact information

### Interactive Elements

- **Project Cards**: Click on any project card to expand and view detailed information
- **Download CV**: Click the "Download CV" button to download the resume PDF
- **External Links**: All external links open in new tabs for easy navigation
